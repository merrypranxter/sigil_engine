# Examples for sigil_engine

This folder contains optional demonstrations you can load in `index.html` by changing the `src` attribute of the `<script>` tag. Each example illustrates a different feature of the engine.

- **random-sigils.js** – Generates a new random statement every few seconds and automatically charges each sigil.
- **burst-demo.js** – Shows only the particle burst effect. Press Space to trigger a burst.
- **strobe-demo.js** – Displays a static sigil and cycles through strobe intensities with the `S` key.

To try one, edit the `<script>` tag in `index.html` to load the desired file instead of `src/js/main.js`.
